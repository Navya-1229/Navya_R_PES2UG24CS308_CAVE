# Navya_R_PES2UG24CS308_CAVE

# 🎓 Seminar Hall Modeling & Unity Interaction Project

## 📌 Project Overview

This project was developed as part of the **CAVE Internship Task**.

The objective was to recreate a **seminar hall environment in Blender** using the provided reference image and implement **interactive features in Unity** using **C# scripting and Unity UI**.

The scene was modeled completely in **Blender without using downloaded assets**, maintaining realistic proportions and modular object design.

---

## 🏛️ Seminar Hall Features (Blender Modeling)

The following elements were created:

✅ Stage
✅ Podium
✅ Central Display Screen
✅ Side Boards
✅ Tiered Seating Arrangement
✅ Tables / Desks
✅ Chairs
✅ Windows
✅ Wall Panels
✅ Ceiling Lights
✅ Ceiling Fans

### ✨ Modeling Highlights

* Realistic seminar hall structure
* Modular asset creation
* Proper object organization and naming
* Basic materials and textures applied
* Entire scene created in Blender

---

## 🎮 Unity Interactions Implemented

### 🖥️ 1. Change Seminar Screen Content

Users can interact with the seminar screen and change presentation content dynamically.

### 🟨 2. Object Highlight Interaction

Objects highlight when clicked, making interaction more immersive and visually clear.

### 🎥 3. Open / Close Presentation Screen Panel

The presentation screen panel can be toggled through interaction.
